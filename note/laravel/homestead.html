# Laravel Homestead Windows安装

# 相关软件

  1. Vagrant 官方网站
  2. Virtual Box 官方网站
  3. Homestead github
  4. Homestead 官方文档

# 本次安装的版本

  1. Homestead 版本 7.20.0
  2. Vagrant Box的Homestead Box版本(box版本不是vagrant版本，本次安装的版本为7.2.0)

# 下载homestead box(下载后名称为virtualbox.box)后放在D:\vagrant文件夹下

# Vagrant安装

下载Windows版本，默认安装

# Virtualbox安装

下载windows版本，默认安装

# Homestead安装

# 创建homestead的box版本配置文件

下载homestead 7.20.0，存放在D盘(视自己情况存放)，在D:\Homestead文件夹创建homestead.json文件，配置文件的url填写下载的box

D:\Homestead\homestead.json

{
    "name": "laravel/homestead",
    "versions": [{
        "version": "7.20.0",
        "providers": [{
        "name": "virtualbox",
        "url": "file:D:\\vagrant\\virtualbox.box"
        }]
    }]
}

# 初始化homestead

在D:\Homestead目录下，打开git bash(或其他命令行工具)，输入:

bash init.sh

结果显示 Homestead initialized! 初始化成功

创建密钥文件，输入：

ssh-keygen

显示

$ ssh-keygen
Generating public/private rsa key pair.
Enter file in which to save the key (/c/Users/Administrator/.ssh/id_rsa):
Enter passphrase (empty for no passphrase):

可以根据自己的情况设置密钥文件保存位置，安装好后可以在对应的位置看到密钥文件

# 添加virtualbox.box

输入：

vagrant box add homestead.json

显示：

$ vagrant box add homestead.json
==> box: Loading metadata for box 'homestead.json'
    box: URL: file://D:/Homestead/homestead.json
==> box: Adding box 'laravel/homestead' (v7.20.0) for provider: virtualbox
    box: Downloading: file:D:\vagrant\virtualbox.box
    box:
==> box: Successfully added box 'laravel/homestead' (v7.20.0) for 'virtualbox'!

# 启动vagrant

输入:

vagrant up

开始安装虚拟机，完成后显示转发端口

# homestead配置

D:\Homestead\Homestead.yaml

---
ip: "192.168.10.10"
memory: 2048
cpus: 1
provider: virtualbox

authorize: ~/.ssh/id_rsa.pub

keys:
    - ~/.ssh/id_rsa

folders:
    - map: ~/code                   # 本机代码目录，~为当前用户的家目录
      to: /home/vagrant/code        # 虚拟机代码目录，虚拟机用户名和密码为vagrant/vagrant

sites:
    - map: homestead.test             # 自己定义的本地域名
      to: /home/vagrant/code/test     # 映射的虚拟机目录，此处要填程序运行目录，如：laravel运行目录在public

databases:
    - homestead   # 数据库连接homestead，用户名为homestead，密码为secret

# hosts文件配置

C:\Windows\System32\drivers\etc\hosts

添加: 192.168.10.10 homestead.test

然后就可以通过浏览器输入域名: homestead.test 进行访问
